# Docyrus UI — React Hooks Catalog

Every hook below ships from `@docyrus/ui`. Each row links to that hook's `llms.txt` page — fetch it (`WebFetch` / curl) before wiring the hook so you have the exact signature, params, and return shape. Install with the CLI: `docyrus add @docyrus/hooks-<slug>` (e.g. `docyrus add @docyrus/hooks-use-docyrus-data-grid`).

## Docyrus-connected hooks (require an authenticated `@docyrus/api-client`)

These hooks fetch from the Docyrus REST API. They typically take a `client` (`@docyrus/api-client`) plus a data-source identifier and return ready-to-spread component props. Pair them with the matching component and, for data formatting, mount `<DocyrusTenantProvider>` near the root (see `useDocyrusTenant`).
